## 定义和用法

## 属性描述
compatMode属性可配置，可枚举，可取值，不可赋值。

```javascript
// Object.getOwnPropertyDescriptor(Document.prototype, 'compatMode') 的结果如下：
var result = {
    configurable: true,
    enumerable: true,
    get: function () { /*[native code]*/ },
    set: undefined
}
```
